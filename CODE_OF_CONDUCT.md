# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

We pledge to make participation in this project a harassment-free experience
for everyone, regardless of background or identity.

## Our Standards

Be respectful and constructive.
Harassment, discrimination, or abusive behavior will not be tolerated.

## Enforcement

Instances of abusive behavior may be reported by opening an issue or contacting
the project maintainers directly.

Maintainers have the right to remove or reject contributions that violate this code.
